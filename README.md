
# yt-dlp-gui

* [yt-dlp](https://github.com/yt-dlp/yt-dlp)のGUI版
    * GUIライブラリはCustomtkinter
* URLを画面上でCtrl+VするだけでDL開始する効率的な仕様
    * 常に最高画質で保存

# インストール

## Windows
denoのインストール
winget install --id=DenoLand.Deno

ffmpegのインストール
https://www.ffmpeg.org/download.htmlからダウンロード

## Mac

denoのインストール
brew install deno

ffmpegのインストール
`brew install ffmpeg`
